Connect the script to the page

To connect the clock to your page, you need to create the container in which it will be displayed and connect the script.
The container is a <canvas> tag.
Pay attention to the script's path! In this case (<script src="./assets/js/MxClock.min.js"></script>), the relative path is used.
You can also use the absolute path on your website. For example, this is correct:
<script src="http://yourwebsite.com/assets/js/MxClock.min.js"></script>

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>Document</title> </head>
<body>

<canvas id="MxClock"></canvas> <!-- Container -->

<script src="./assets/js/MxClock.min.js"></script> <!-- script -->
</body>
</html>

The next step is to call the script:

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>Document</title> </head>
<body>

<canvas id="MxClock"></canvas> <!-- Container -->

<script src="./assets/js/MxClock.min.js"></script> <!-- script -->

<script>
new CreateClock( '#MxClock' );
</script>
</body>
</html>

Customizable appearance

You can customize the style of the clock to your liking. The second function argument is the transfer of the property object.
Below is a list of properties that you can use to set up your watch.

new CreateClock( '#MxClock', {
sizeClock: 400, // Clock size 'integer or decimal'
moveToCenter: 3.4, // Change the location of the rooms to the center 'integer or decimal'
backgroundColor: '#003366', // Background color 'string'
centerCircle: 22.5, // The size of the central circle 'integer or decimal'
colorCenterCircle: '#ffd308', // The color of the central circle 'string'

// gradient for arrows
gradientCenter: 'orange', // The color of the arrows. The gradient begins at the center 'string'
gradientOutside: 'blue', // The color of the arrows. The gradient begins outside 'string'

// numbers
fontSizeNumbers: 25, // Font number size 'integer or decimal'
colorNumbers: '#6699CC', // Color of numbers 'string'

// seconds dots
colorSecondsDots: '#6699CC', // Color of seconds points 'string'
sizeSecondsDots: 1, // Size of seconds points 'integer or decimal'

// arrows
hourArrowSize: 40, // An hour's arrow'integer or decimal'
minuteArrowSize: 50, // An arrow of a minute 'integer or decimal'
secondArrowSize: 60, // An arrow of a second 'integer or decimal'
} );
